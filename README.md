# 我的在线记事本 📝

一个极简的个人记事本应用，让你可以随时在网页上记录和查看文字内容。

## ✨ 功能特点
- 创建笔记并保存到云端数据库
- 查看所有笔记列表
- 简洁美观的界面
- 数据永久保存

## 🚀 快速开始
1. 访问：https://my-online-notebook.vercel.app
2. 在输入框中写下笔记
3. 点击"保存笔记"
4. 笔记会自动保存到云端

## 🛠️ 技术栈
- 前端：HTML, CSS, JavaScript
- 后端：Node.js, Express
- 数据库：MongoDB Atlas
- 部署：Vercel
